# Python Study Order (Foundations to AI/ML)

Use [MASTER_INDEX.md](MASTER_INDEX.md) for concept-to-file mapping, short descriptions, and difficulty levels.

## Phase 1: Python Fundamentals (Syntax to OOP)

Follow folders in this sequence:

1. `01_syntax_basics`
2. `02_control_flow`
3. `03_data_structures`
4. `04_functions_modules`
5. `05_file_handling`
6. `06_oops_fundamentals`
7. `07_oops_advanced`
8. `08_visualization`
9. `09_practice_problems`

## Phase 2: AI/ML Foundations (NumPy to Scikit-Learn)

Continue with folders in this sequence:

10. `10_numpy_basics` (2-3 hours) - Foundation for all numerical computing
11. `11_pandas_basics` (3-4 hours) - Essential for data manipulation and EDA
12. `12_matplotlib_seaborn` (2-3 hours) - Exploratory data analysis visualization
13. `13_math_for_ml` (3-4 hours) - Mathematical foundations (linear algebra, stats, probability)
14. `14_sklearn_basics` (4-5 hours) - Scikit-learn algorithms and model training
15. `15_ml_workflow` (2-3 hours) - Best practices, pipelines, and deployment

## How to revise quickly (Phase 1)

- Read the `# Revision Notes` at the top of each file.
- Predict output before running each script.
- Change one line/input and rerun to observe behavior.
- Finish one folder before moving to the next.

## How to study Phase 2

- Each file is a comprehensive reference: ~800-1000 lines of code
- Study 1-2 files per day
- Run all examples and modify code to understand WHY each concept matters
- Use inline comments as your learning guide
- Study MATH concepts (section 13) before advanced ML algorithms (section 14)
- Complete end-to-end workflows (section 15) after learning individual algorithms

## Suggested daily flow (Phase 1)

1. Run 2 files from current folder.
2. Explain the concept in your own words.
3. Solve 2 files from `09_practice_problems`.
4. Revisit mistakes and add your own comments.

## Suggested daily flow (Phase 2)

1. Read one file completely from top to bottom.
2. Run all code examples and observe outputs.
3. Modify 3-5 lines to test your understanding.
4. Summarize the KEY TAKEAWAY in your own words.
5. (Optional) Write your own small example combining concepts.
